## Escape Sequences

| Escape Sequence | Represents        |
| :-------------: | :---------------- |
|       \0        | Null              |
|       \n        | New Line          |
|       \t        | Tab (Horizontal)  |
|       \v        | Tab (Vertical)    |
|       \\'       | Single Quote      |
|       \\"       | Double Quote      |
|       \\\       | Backslash         |
|       \\?       | Question Mark     |
|       \b        | Backspace         |
|       \a        | Alarm (Beep/Bell) |
|       \r        | Carriage Return   |
